**Option A: Start from Template**
```
1. Browse "Workflow Templates"
2. Click any template card

4. Customize as needed
```

**Option B: Create from Scratch**
```
1. Click "Create New File"
2. Empty canvas opens
3. Add nodes from sidebar
4. Connect them together
```

## Step 5: Connect Nodes

**How Connections Work:**

1. Each node has **yellow dots** (outputs) and **cyan dots** (inputs)
2. **Click and drag** from yellow dot
3. **Connect to** cyan dot on another node
4. **Release** to create connection
5. Line appears and saves automatically
